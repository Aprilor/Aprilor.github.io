浏览器 内核

HTML

标签

标签 描述
<!–…–> 定义注释
<!DOCTYPE> 定义文档类型
<a> 定义锚
<abbr> 定义缩写
<acronym> 定义首字母缩写
<address> 定义文档作者或拥有者的联系信息
<area> 定义图像映射内部的区域
<b> 定义粗体字
<blockquote> 定义从其他来源引用的节
<body> 定义文档的主体
<button> 定义按钮
<caption> 定义表格标题
<cite> 定义著作的标题
<code> 定义计算机代码文本
<dd> 定义定义的描述
<del> 定义被删除文本
<div> 定义文档中的节或区域（块级）
<dl> 定义定义列表
<dt> 定义定义项目
<em> 定义强调文本
<form> 定义供用户输入的 HTML 表单
<h1> to <h6> 定义 HTML 标题
<head> 定义关于文档的信息
<html> 定义 HTML 文档
<i> 定义斜体字
<iframe> 定义内联的子窗口（框架）
<li> 定义列表的项目
<link> 定义文档与外部资源的关系
<input> 定义输入控件
<ol> 定义有序列表
<p> 定义段落
<pre> 定义预格式文本
<q> 定义短的行内引用
<samp> 定义计算机代码示例
<script> 定义客户端脚本
<span> 定义文档中的行内的小块或区域
<strong> 定义强调文本
<style> 定义文档的样式信息
<sub> 定义下标文本
<sup> 定义上标文本
<table> 定义表格
<tbody> 定义表格中的主体内容
<td> 定义表格中的单元
<tfoot> 定义表格中的表注内容（脚注）
<th> 定义表格中的表头单元格
<thead> 定义表格中的表头内容
<title> 定义文档的标题
<tr> 定义表格中的行
<ul> 定义无序列表
<var> 定义变量
单标签 描述
<br /> 定义简单的折行
<hr /> 定义水平线
<img /> 定义图像
<meta> 定义关于 HTML 文档的元信息

容器

标签 效果
div 独占一行
span 没效果

标签属性

标签 属性 属性值 描述 例子
<a> herf url 链接或锚点 <a href=”http://www.w3school.com.cn">This is a link</a>
<a> target _blank 新窗口打开文档 <a href=”http://www.w3school.com.cn/" target=”_blank”>Visit W3School!</a>
<img> src url 源属性 <img src=”url” />
<img> alt 替换文本属性 <img src=”boat.gif” alt=”Big Boat”>
